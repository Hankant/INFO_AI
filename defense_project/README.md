# Defense Project Package

This folder contains the English LaTeX translation of the proposal, a BibTeX bibliography, and a dedicated directory for reference materials.

## Structure

- `latex/main.tex`: main LaTeX file
- `latex/sections/`: translated section files
- `latex/references.bib`: BibTeX bibliography
- `reference_materials/`: place to store PDFs, notes, and other reference files for the defense

## Notes

- The section numbering follows the source PDF as closely as possible, including the source draft's numbering gaps.
- A small number of bibliography entries are future-dated or internally inconsistent in the source draft. Those entries are preserved in `references.bib` with the best verifiable metadata available and a note where manual checking may still be useful.
